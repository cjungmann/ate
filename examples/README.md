# COMPANION DIRECTORY FOR ATE-EXAMPLES(7) (man page)

As I start working on the **ate-examples** man page, I am also making
a plan for managing the complexity.  Since the main advantage of using
this tool is in managing large datasets, it is helpful to demonstrate
using large datasets as well.

It's hard to create large tables out of thin air.  The examples in
this directory uses data found at [openintro][openintro_inventory] or
at [suggestions][suggestions]
